Punto de venta para purificadoras de agua

Este proyecto está implementado en una aplicación de escritorio para windows, hecha en C#.

Básicamente es un CRUD, es decir, que tiene conexión a una base de datos, a la cual se le pueden hacer las 4 acciones básicas (dar de alta, consultar, actualizar y dar de baja).

Esta aplicación ofrece funciones y herramientas para registrar ventas, registrar datos de clientes, etc. Así como opciones para visualizar la entrada, y salida de dinero con el fin de proporcionar una alta gama de opciones para poder administrar un negocio como lo es una purificadora.

Este sistema, se encuentra actualmente en producción, y sigo trabajando en él constantemente para darle mantenimiento y añadirle características.
